# 📖 AI Novel Studio — 产品需求文档

> 版本：v1.0 | 日期：2026-06-14 | 状态：待老板确认

---

## 一、产品定位

**一句话**：AI 辅助网文创作工具，帮作者从设定到成稿全流程提效。

**目标用户**：网文作者（起点/番茄/七猫等平台的写手），海外英文网文作者。

**核心卖点**：免费额度 + 多 AI 可切换 + 已有小说续写 + 桌面软件离线使用。

---

## 二、功能清单（已实现 ✅ / 计划中 🔲）

### 2.1 世界观与角色设定 ✅

| 功能 | 说明 |
|------|------|
| 世界观选择 | 西幻/修仙/现代/末世，支持自定义描述 |
| 主角设定 | 姓名/性格/能力/弱点，支持随机生成 |
| 正反派设定 | 无限添加角色，随机阵容生成 |
| 故事风格 | 8 种预设风格 + 自定义 |
| 篇幅选择 | 短篇(~20章)/中篇(~40章)/长篇(~60章)/超长篇(~80章) |
| 应用框架 | 一键重置所有生成内容，基于新设定重新来 |

### 2.2 AI 生成流程 ✅

| 步骤 | 功能 | 说明 |
|------|------|------|
| 1 | 开篇点子 | AI 生成 5 个开篇方向，可无限追加 |
| 2 | 编辑+优化 | 手动改点子，AI 再优化推荐 |
| 3 | 生成大纲 | 按篇幅自动生成对应章数大纲 |
| 4 | 生成章节 | 选择章节，AI 写出完整正文 |
| 5 | 保存大纲 | 已写章节存回大纲，回头可查看 |
| 6 | 下一章 | 自动弹出续写方向，AI 按方向续写 |
| 7 | 故事追踪 | AI 盘点角色状态/伏笔/关系 |

### 2.3 已有小说续写 ✅

| 功能 | 说明 |
|------|------|
| 粘贴/上传 | 支持直接粘贴全文或上传 .txt |
| AI 分析 | 自动提取世界观/主角/角色/剧情/当前情境 |
| 同步框架 | 一键填充左面板所有设定 |
| 续写 | 从分析结果继续写后续章节 |

### 2.4 AI 平台 ✅

| 平台 | 费用 | 适用场景 |
|------|------|----------|
| DeepSeek | 注册送额度 | 国内用户 |
| 智谱 GLM | 注册送额度 | 备选 |
| Agnes AI | 免费 | 海外用户 |
| 自定义 Key | 用户自带 | 测试/高级用户 |

### 2.5 其他已实现 ✅

| 功能 | 说明 |
|------|------|
| 中英文切换 | 一键切换全部界面语言 |
| 免费额度 | 当前 999 次/天（测试），后续改付费 |
| 市场调研 | AI 分析网文市场 8 维度 |
| 章节审核 | 检查角色一致性 |
| 导出/导入 | .json 格式备份项目 |
| 移动端 | 手机浏览器可用 |
| 面板拖拽 | 切换按钮可拖到任意位置 |

### 2.6 计划中 🔲

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 桌面应用 (.exe) | 🔴 高 | Electron 打包，双击即用 |
| 付费系统 | 🔴 高 | 月卡/按次/永久 |
| 用户系统 | 🟡 中 | 注册/登录/云端存储 |
| 多项目管理 | 🟡 中 | 同时写多本书 |
| 写作数据统计 | 🟢 低 | 字数/速度/习惯分析 |
| 社区/模板 | 🟢 低 | 预设模板市场 |

## 三、商业模式（建议）

| 方案 | 价格 | 内容 |
|------|------|------|
| 免费版 | ¥0 | 3 次/天，Agnes AI，导出水印 |
| 基础版 | ¥19/月 | 100 次/天，DeepSeek，无水印 |
| 专业版 | ¥49/月 | 无限次，全 AI，云端存储，优先支持 |
| 永久版 | ¥299 | 买断，无限使用 |

**早期策略**：先免费拉用户，有 1000 用户后上付费。

## 四、技术架构

```
ai-novel/
├── app/                    # Next.js 前端页面
│   ├── page.js             # 主页面（核心）
│   ├── api/generate/       # AI 接口
│   └── database/           # 数据库页面
├── electron/               # 桌面应用（新增）
│   ├── main.js             # Electron 主进程
│   └── preload.js          # 预加载脚本
├── package.json            # 项目配置
└── .env.local              # 本地密钥（不提交）
```

- 前端：Next.js 16 + React 19 + Tailwind CSS 4
- AI 接口：OpenAI 兼容格式，支持 3 平台
- 桌面：Electron（计划）
- 部署：Vercel（网页）/ 本地 .exe（桌面）
